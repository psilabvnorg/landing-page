# PSI Technology - GitHub Pages Site

A simple, clean website showcasing PSI Technology and team member profiles.

## Setup Instructions

1. **Create a new GitHub repository** (or use existing one)
2. **Upload all files** to your repository:
   - `index.html` (main page)
   - `styles.css` (styling)
   - `cuong.html` (Cuong's profile)
   - `duy.html` (Duy's profile) 
   - `tung.html` (Tung's profile)
   - `company_logo.png` (your logo)

3. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click Save

4. **Access your site**:
   - Your site will be available at: `https://[username].github.io/[repository-name]`
   - It may take a few minutes to deploy

## Features

- **Responsive design** - works on desktop and mobile
- **Clean, professional layout**
- **Individual profile pages** for each team member
- **Project portfolio showcase**
- **Contact information**
- **Easy navigation** between pages

## Customization

- Replace `company_logo.png` with your actual logo
- Update contact information in `index.html`
- Modify colors in `styles.css` if needed
- Add more team members by creating new HTML files
